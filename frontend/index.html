下载教程

先看上面的“本机配置”。如果使用 TensorRT/CUDA 后端，请保证电脑是 NVIDIA 显卡，并且显卡驱动支持你下载的 CUDA 12.x 版本。

带“下载/继续”的项目会直接下载到程序目录的 downloads\。未完成文件会保留为 .part，下次点击同一按钮会继续下载。

ONNX Runtime
在 GitHub Releases 下载 Windows x64 GPU 包，文件名通常类似 onnxruntime-win-x64-gpu-*.zip；CPU 后端用 onnxruntime-win-x64-*.zip。
CUDA
进入 CUDA Toolkit Archive，选择 Windows x86_64 的 CUDA 12.x；解压或安装后把 bin 目录配置进来。
cuDNN
下载 cuDNN 9.x for CUDA 12.x 的 Windows x86_64 包，把所有 cudnn*.dll 放在同一目录。
TensorRT
下载 TensorRT 10.x Windows x64 zip，选择匹配 CUDA 12.x 的版本；把 lib 或 bin 目录配置进来。
OpenCV
下载 Windows 版本后，选择包含 opencv_world*.dll 的目录。
放置方式
最省事是把下载到的 DLL 放进 backend\；也可以用每张缺失卡片里的“选择位置”单独指定目录。
